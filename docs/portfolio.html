"Dear Data" Project

"Dear Data" is a year-long, analog data drawing project by Giorgia Lupi and Stefanie Posavec that we (DIGIT 100 with Dr. Beshero-Bondar) studied in the 2024 Fall semester at Penn State Behrend. After studying the project we then developed our own human data visualizations that yielded tremendous variety. The purpose of this project was to educate ourselves on the different ways data can be presented and interpreted by different people.
